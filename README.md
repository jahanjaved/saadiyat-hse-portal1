# Saadiyat Lagoons HSE Workbook Portal

This package shows all workbook sheets on one website page.

## Files
- index.html
- assets/styles.css
- scripts/app.js
- data/workbook-data.json
- source/<original workbook>

## GitHub Pages
Use:
- Deploy from branch
- main
- / (root)

## Important
This version is static and already contains the converted workbook data.
If you update the Excel file later, the JSON file will need to be regenerated and replaced.
