# Saadiyat Lagoons HSE Website

This is a full static website package built from the latest workbook JSON export.

## What is included
- Professional multi-page website
- All JSON data from the latest Excel workbook export
- Dashboard chart images
- CSV backups
- All sheet JSON files
- GitHub-ready file structure

## Main pages
- `index.html`
- `dashboard.html`
- `inspections.html`
- `capa.html`
- `gap-analysis.html`
- `calendars.html`
- `data-explorer.html`

## How to use
1. Extract the ZIP file.
2. Open `index.html` directly for a quick preview, or upload the full folder to GitHub Pages.
3. Later, when your Excel changes, replace the files inside the `data` folder with fresh JSON export files and refresh the site.

## Important note
A few source workbook fields still have broken Excel references (`#REF!`) in the original workbook. This website does not invent missing values; it shows the exported workbook data faithfully.
