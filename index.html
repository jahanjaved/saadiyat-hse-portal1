Recommended design logic

One page, but not one style for all sheets.

Dashboard-type sheets should be shown as KPI cards and summary panels. Data-heavy sheets should be shown as readable tables with search, filters, and sticky headers. This keeps the website professional while still respecting the exact workbook structure.

Best practice: keep the exact Excel sheet names in the menu and section IDs, but present the data inside each section using the most readable format for that sheet.
